## Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

### Navigate into the project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Start the server

```bash
npm run dev
```
